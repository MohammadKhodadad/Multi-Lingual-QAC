\begin{abstract}
We present a benchmark-construction pipeline for multilingual legal retrieval using the JRC-Acquis corpus. The benchmark treats \texttt{CELEX} as a stable cross-lingual document anchor, builds one retrieval document per \((\texttt{CELEX}, \texttt{language})\) realization, and derives multilingual retrieval supervision through same-language legal question--answer generation. The construction flow separates retrieval context from generation context, filters source documents through legal-substance heuristics, generates one focused legal information need from a selected target-language document realization, and links each accepted query to the retained sampled documents that share the same \texttt{CELEX}. We package the result in standard retrieval form---\texttt{corpus}, \texttt{queries}, \texttt{qrels}, and \texttt{qac}---and evaluate it through a Hugging Face-backed MTEB workflow with multilingual sentence-embedding baselines.
\end{abstract}
